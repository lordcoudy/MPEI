ЛР8 - работа ООП C#.
Задание: Разработать ОО программу моделирования заданной предметной области.
"Глубина" иерархии - не менее 4, "ширина" - от 3-х.

При разработке программы учитывается: использование делегатов, интерфейсов и пр.
Реализовать для иерархии механизм интерфейсов, один из классов должен реализовывать как минимум два интерфейса. Использовать для проверки всех методов данного класса многоадресный делегат.
Реализовать обработку ошибок (переопределив с помощью наследования одно из событий: StackOverflowException, ArrayTypeMismatchException, DivideByZeroException, IndexOutOfRangeException, InvalidCastException,
OutOfMemoryException, OverflowException).
Показать пример использования полиморфизма.

Отчет по ЛР ООП должен содержать:
   Задание
   Описание предметной области (если есть, то формулы и пр.)
   Разработка: диаграмма классов с описание полей и методов, пользовательский интерфейс, алгоритмы
   Реализация (описание выполнения задания), результаты выполнения (тестирование)
   Код программы
   Выводы
