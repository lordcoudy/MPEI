RecvAdr SendAdr - устройство для последовательной передачи 8 бит данных по адресу 4 бит.
3.pbs - тестовый проект с 4-я абонентами.

Назначение прерываний:
INT0 - отправка данных (порт C) по адресу (порт D); 
INT1 - прием данных, если данные адресуются нам; 
INT2 - -; 
INT3 - -;

На порт B выводится принятый байт по прерыванию INT1.
