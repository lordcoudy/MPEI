Вариант 15. Разработать шаблон автоматного компонента, имеющего 3  входа и 1 
выход, выполняющего следующую задачу. 
В каждом сеансе во входных буферах на информационных входах X1, X2 компонента, 
функционирующего по разрабатываемому шаблону, лежат по N вещественных чисел, 
упорядоченных по возрастанию. Значение N может меняться в начале следующего сеанса 
если есть данные на входе X3. Если данных на входе X3 на начало очередного сеанса нет, то 
используется предыдущее значение  N.
Основной режим работы компонента состоит в объединении двух упорядоченных по 
возрастанию последовательностей, и выдаче на выход Y1 объединенной упорядоченной 
последовательности  из 2N  чисел в каждом сеансе.  В конце сеанса необходимо также 
вывести на выход Y1 метку об окончании текущего сеанса.
Нарисовать диаграмму переходов и представить описание шаблона на языке 
представления асинхронных автоматных схем при заданных рабочих ситуациях на 
входах компонента в начальном состоянии S0:
State S0: U1={111}, U2={100}. U3={011}.

f0 = считываю x3, N = x3, nextState = SS

f1 = считываю x1, x2, вывод min(x1, x2), в память max(x1, x2), nextState = S2 or S1 (зависит от x1 и x2)

f2 = считываю x1, вывод min(x1, память), в память max(x1, память), if (i < 2N) then nextState = S2 or S1 (зависит от x1 и x2) else nextState = S3

f3 = считываю x2, вывод min(память, x2), в память max(память, x2), if (i < 2N) then nextState = S2 or S1 (зависит от x1 и x2) else nextState = S3

f10 = считываю x2, вывод x2, if (i = 2N) then nextState = S3

f11 = считываю x1, вывод x1, if (i = 2N) then nextState = S3

f4 = из памяти N, nextState = SS

f5 = считываю x3, N = x3, nextState = SS

f6 = считываю x2, x3, nextState = S4

f7 = считываю x1, nextState = SS

f8 = считываю x1, nextState = S5

f9 = считываю x2, x3, nextState = SS
